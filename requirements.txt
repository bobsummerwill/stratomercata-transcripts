# ==============================================================================
# WhisperX Requirements - Hardware-Agnostic
# ==============================================================================
#
# This project now uses split requirements files for different hardware:
#
# - requirements-nvidia.txt  → PyTorch nightly with CUDA (for NVIDIA GPUs)
# - requirements-cpu.txt     → PyTorch nightly CPU-only (for non-NVIDIA systems)
# - requirements-base.txt    → Common packages (whisperx, pyannote, speechbrain)
#
# The install_packages_and_venv.sh script automatically detects your hardware
# and installs the appropriate PyTorch version.
#
# ==============================================================================

# AUTOMATED INSTALLATION (RECOMMENDED):
#   ./install_packages_and_venv.sh
#
# MANUAL INSTALLATION:
#   For NVIDIA GPUs:
#     pip install -r requirements-nvidia.txt
#     pip install -r requirements-base.txt
#
#   For CPU-only systems:
#     pip install -r requirements-cpu.txt
#     pip install -r requirements-base.txt
#
# For locked versions: see requirements-lock.txt
# For setup guide: see SETUP_RTX_5070.md
#
# ==============================================================================
